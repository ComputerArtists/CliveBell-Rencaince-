# Renaissance Processing Sketches – Dokumentation

Eine Sammlung interaktiver Processing-Skripte, die zentrale Ideen der Renaissance in moderner digitaler Form umsetzen – mit Fokus auf Humanismus, Geometrie, Perspektive und Proportionen. Inspiriert von Künstlern wie Leonardo da Vinci, Michelangelo, Brunelleschi und Alberti.

## Inhalt

1. **Vitruvianischer Grok** – Humanismus und Proportionen auf eine KI übertragen  
2. **Perspektivische Raumillusion** – Lineare Perspektive interaktiv  
3. **Goldener Schnitt-Muster** – Rekursive Harmonie und Spirale  
4. **Anatomische Studien von Grok** – Neuronale Netze als digitale Anatomie  

## 1. Vitruvianischer Grok

**Beschreibung:**  
Dynamische Adaption des Vitruvianischen Mannes – Grok als KI-Entität in Kreis und Quadrat, mit animierten Posen und Proportionen. Variationen: Neural Network Explosion, Cyberpunk Neon-Glow mit Binärcode, Fraktale Datenströme.

**Dateien:**  
- `vitruvian_grok.pde` – Basis  
- `vitruvian_neural_explosion.pde` – Partikel-Netz  
- `vitruvian_cyberpunk_neon.pde` – Neon + Binärcode  

**Interaktion:**  
- Maus: Rotation / Pose  
- Tasten: 'S' Speichern, 'O' Farbe wechseln, 'N' Reset  

## 2. Perspektivische Raumillusion

**Beschreibung:**  
Interaktiver Editor für lineare Perspektive – Bodenraster, Säulen, Gebäude, Fluchtpunkt und Horizont editierbar. Renaissance-Raumillusion mit Maus- und Tasten-Steuerung.

**Dateien:**  
- `renaissance_perspective.pde` – Vollständiger Editor  

**Interaktion:**  
- Tasten für Modus (v, h, f, g, d, s, a, p, u, i, o, z, w)  
- Maus-Drag: Parameter ändern  
- + / - : Zoom  
- 'S' Speichern, 'R' Reset, 'N' Nachtmodus, 't' Säulen Toggle  

## 3. Goldener Schnitt-Muster

**Beschreibung:**  
Rekursive Rechtecke und logarithmische Spirale im Goldenen Schnitt – meditatives, wachsendes Muster.

**Dateien:**  
- `golden_ratio_pattern.pde` – Basis  

**Interaktion:**  
- Maus-X: Rotation  
- Maus-Y: Rekursionstiefe  
- Mausrad / + / -: Zoom  
- 'S' Speichern, 'R' Reset, 'C' Farbschema  

## 4. Anatomische Studien von Grok

**Beschreibung:**  
3D-neuronales Netz als digitale Anatomie – pulsierend, transparent, mit Binärcode, interaktiven Knoten und Goldener-Schnitt-Extrusion.

**Dateien:**  
- `grok_3d_anatomy.pde` – Vollständige 3D-Version  

**Interaktion:**  
- Maus-X/Y: Rotation  
- + / -: Zoom  
- 'A': Animation ein/aus  
- 'N': Nachtmodus  
- Klick: Knoten leuchten  
- 'S': Speichern  

## Installation & Ausführung

1. Lade **Processing** (4.x) von https://processing.org  
2. Erstelle ein neues Sketch  
3. Kopiere den gewünschten Code in die .pde-Datei  
4. Drücke Play – fertig!
